# SMAI Assignment 3: Indian Street Food Recognizer

This repository contains the implementation of a real-time ML application that recognizes Indian street food, estimates calorie counts, and identifies allergens.

## Project Overview
- **Course:** Statistical Methods in AI (SMAI) - IIIT Hyderabad
- **Objective:** Build a working web demo for Indian food recognition.
- **Theme:** T2 — Indian Food Recognizer (Variant: Street Food)
- **Tech Stack:** Python, PyTorch/TensorFlow, Streamlit, Kaggle Datasets.

## Features
- [ ] Image classification for 12+ street food variants.
- [ ] Calorie estimation based on dish identification.
- [ ] Allergen flagging (Nuts, Dairy, Gluten).
- [ ] Interactive Streamlit/Gradio UI.

## Repository Structure
- `documentation/`: Detailed project docs and technical reports.
- `notebooks/`: Training and data analysis Jupyter notebooks.
- `app/`: Source code for the web application.
- `requirements.txt`: Python dependencies.
